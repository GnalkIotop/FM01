FM Drum Firmware Update

Verbinde dein Geraet per USB-Datenkabel und klicke auf Install.

Schritte

  1. Chrome oder Edge verwenden.
  2. USB-Datenkabel anschliessen.
  3. Auf Install klicken und den richtigen Port waehlen.
  4. Warten, bis das Update abgeschlossen ist.

Wenn kein Port erscheint

  1. BOOT gedrueckt halten.
  2. Kurz RESET tippen.
  3. BOOT loslassen und erneut verbinden.
